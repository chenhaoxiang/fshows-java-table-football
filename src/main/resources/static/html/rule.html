桌上足球比赛规则

三条基本规则

每局先进10球者获胜；
一个合乎规则的进球计一分（即使球进入球门后被弹出来，也同样计分）；
击球之前或之后旋转球杆超过360°视为犯规。

一些比较详细的简单比赛规则

1、猜球决定优先权，胜者选择首局站位和先发球，从发球口发球；
2、发球后必须由小人碰到球后形成的进球才能计分；
3、进球后，被进球一方发球；
4、如球被击出球台，上次被进球一方发球；
5、球停在双方小人无法碰到的区间被称为“死球”，如果后场死球，由守门员重新发球。其它地方死球由上一次被进球一方发球；
6、正在控球一方将球停止后，有权叫暂停，每局每方有两次暂停，每次不超过30秒；
7、比赛状态时，比赛双方不得在球台上用手去触球；
8、比赛间不允许有不文明言行；
9、比赛时，控球方持球最多不能超过12秒。
10、记分： A、任何合法的进球都算得分（包括射进球门后反弹出来）
B、如果双方球队都认为已进球，但没有加分，且没有其他得分时，那么在双方都认可的情况下它应该补加分。
11、旋转球杆：
在击球前后旋转球杆超过360度属犯规（无故旋转球杆会被判为故意扰乱对方注意力）。
处罚：A、如果旋转球杆将球打进，判为不得分，由对方守门员发球。
如不进球，对方可以继续比赛或重新发球。
12、撞动球台：
任何晃动，抬起或移动球台造成对方控球不利都称为撞动球台。
处罚：A、第一、二次，提出警告，或在撞击初始位置或当前位置继续比赛，或重新发球。
第三次，判为技术犯规并且按A项列出选择重新进行比赛。
13、故意扰乱对方注意力
可能影响选手与操作球台无关的攒动或声音（除小人击球时产生的声音外）。
处罚：A、如由进攻方造成：判进球不得分，由对手重新发球。
B、其他情况：受影响的一方选择继续比赛或重新发球
14、技术犯规：
公然或故意犯规被判为技术犯规。注：在一次技术犯规后，同队球员可以交换站位（双打）。
处罚：A、立即停止比赛
B、一次由三人杆射门的机会
C、从停止点（无论得分与否），重新开始。
D、每场比赛无论打到哪一局，三次累积技术犯规将会取消本场比赛。
15、一局结束，双方互换场地。
每场比赛选择出一位裁判。
未在规则中列出的，由裁判决定！
